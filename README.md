📌 Project Overview

This project analyzes how phone addiction and lifestyle factors affect student productivity using machine learning.

📂 Dataset
10,000 student records
Features include study hours, screen time, sleep, focus, etc.
⚙️ Techniques Used
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Machine Learning Models
🤖 Models Used
Random Forest (Best Model)
Linear Regression
KNN
Decision Tree
📈 Results
Best R² Score: 0.76 (Random Forest)
